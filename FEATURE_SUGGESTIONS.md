# LMS 시스템 추가 기능 제안

현재 구현된 기능을 기반으로 추가하면 좋을 기능들을 제안합니다.

## 🎯 우선순위 높은 기능

### 1. **성적 관리 시스템** ⭐⭐⭐
- **필요성**: 학생의 학업 성과를 관리하고 추적
- **기능**:
  - 중간고사, 기말고사, 과제, 출석 점수 입력
  - 자동 총점 계산 및 등급 부여 (A+, A, B+, B, C+, C, D+, D, F)
  - 평점(GPA) 자동 계산
  - 학생별 성적표 조회
  - 강의별 성적 통계

### 2. **출석 관리 시스템** ⭐⭐⭐
- **필요성**: 학생 출석 현황 관리
- **기능**:
  - 일별 출석 체크 (출석, 결석, 지각, 공결)
  - 출석률 자동 계산
  - 출석 점수 반영
  - 출석 통계 및 리포트

### 3. **공지사항 시스템** ⭐⭐
- **필요성**: 강의 관련 공지사항 전달
- **기능**:
  - 강의별 공지사항 작성
  - 전체 공지사항
  - 중요 공지 상단 고정
  - 알림 기능 연동

### 4. **강의 자료 관리** ⭐⭐
- **필요성**: 강의 자료 업로드 및 공유
- **기능**:
  - 파일 업로드 (PDF, PPT, DOC 등)
  - 자료 카테고리 분류
  - 다운로드 통계
  - 자료 검색

### 5. **과제 관리 시스템** ⭐⭐⭐
- **필요성**: 과제 제출 및 평가
- **기능**:
  - 과제 생성 및 마감일 설정
  - 학생 과제 제출
  - 파일 첨부 지원
  - 점수 입력 및 피드백
  - 제출 현황 조회

## 📊 중간 우선순위 기능

### 6. **시간표 관리** ⭐⭐
- **필요성**: 강의 시간 및 장소 관리
- **기능**:
  - 요일별 시간표 설정
  - 강의실 정보
  - 시간표 충돌 체크
  - 학생 개인 시간표 조회

### 7. **학기 관리** ⭐⭐
- **필요성**: 학기별 데이터 분리 및 관리
- **기능**:
  - 학기 생성 및 관리
  - 현재 학기 설정
  - 학기별 강의 조회
  - 학기별 성적 조회

### 8. **선수과목 체크** ⭐
- **필요성**: 수강신청 시 선수과목 이수 여부 확인
- **기능**:
  - 강의별 선수과목 설정
  - 수강신청 시 자동 체크
  - 선수과목 미이수 시 경고/차단

### 9. **알림 시스템** ⭐⭐
- **필요성**: 중요한 이벤트 알림
- **기능**:
  - 수강신청 승인/반려 알림
  - 성적 공지 알림
  - 과제 마감 알림
  - 공지사항 알림
  - 읽음/안읽음 표시

### 10. **질문/답변 (Q&A)** ⭐
- **필요성**: 학생-강사 간 소통
- **기능**:
  - 강의별 질문 작성
  - 강사 답변
  - 익명 질문 옵션
  - 질문 검색

## 🔍 개선 기능

### 11. **검색 및 필터링** ⭐⭐
- 강의 검색 (강의명, 강의코드, 강사명)
- 상태별 필터링 (모집중, 정원마감 등)
- 학점별 필터링
- 학년별 필터링

### 12. **대시보드** ⭐⭐
- 학생 대시보드: 내 수강신청, 성적, 출석률, 과제 현황
- 강사 대시보드: 내 강의, 수강신청 현황, 성적 입력 현황
- 관리자 대시보드: 전체 통계, 사용자 관리

### 13. **통계 및 리포트** ⭐
- 학생별 성적 통계
- 강의별 통계
- 학기별 통계
- 출석률 통계
- Excel/PDF 리포트 생성

### 14. **학년별 최대 학점 제한** ⭐
- 1학년: 최대 15학점
- 2-4학년: 최대 18학점
- 수강신청 시 자동 체크

### 15. **파일 첨부 기능** ⭐⭐
- 프로필 사진 업로드
- 과제 파일 첨부
- 강의 자료 업로드
- 파일 저장소 관리

- 시험 일정 캘린더

### 17. **메시징 시스템**

- 스트리밍
- 시청률 추적

### 19. **온라인 시험**
- 객관식/주관식 문제
- 자동 채점
- 시간 제한
- 부정행위 방지

### 20. **포트폴리오**
- 학생 포트폴리오 관리
- 프로젝트 업로드
- 성과 공유
